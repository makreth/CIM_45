Resources

  • W3Schools - My Teacher

    W3Schools is the largest web developer site. Well, I assume it to be, since it says it on the tin. This is where I got most of my hobbyist programming information from. I've programmed pretty cool website apps just from this site, and it is my primary reference for HTML and CSS. However, since I'm a W3Schools baby, most of my knowledge is spotty, incohesive, and lacking in many areas that don't revolve around pretty styling or cool javascript tricks.

  • Stack Overflow - My Crutch

    Stack Overflow has become a programming meme of sorts. I've seen a funny video about expectations vs. reality with programming. Programmers have been glorified by movies as awesome hacking ninjas. Though those ninjas definitely exist, in reality, programmers can find themselves in a loop of hitting run, sighing in disappointment at red text, going to stack overflow, fiddling with the code, and repeating that process until it works. I eventually want to get to the point where I can hack/program with awesome synthwave music in the background, plus the shades and the matrix outfit, but I'm a stack overflow plebian at the moment.

  • REPL.IT - A Cool Online Programming Tool

    repl.it is a nice online compiler for multiple languages, such as Python, Objective-C, Java, etc. It does HTML/CSS/ Javascript on the fly as well, and it also immediately hosts all created webpages on repl's server, so it can be shared with anyone.
